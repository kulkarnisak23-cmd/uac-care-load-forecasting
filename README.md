# Predictive Forecasting of Care Load & Placement Demand

## Overview
This project predicts future HHS care load and discharge demand using machine learning and time-series forecasting.

## Models Used
- ARIMA
- Random Forest
- Gradient Boosting

## Features
- Forecasting dashboard
- KPI metrics
- Interactive visualizations
- Streamlit deployment

## Run Locally

Install requirements:
```bash
pip install -r requirements.txt
```

Run dashboard:
```bash
streamlit run app.py
```
